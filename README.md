# Sales Data Analysis with Pandas & Matplotlib

This project analyzes a real-world online sales dataset using Python. It focuses on identifying top-selling products and high-value customers through data cleaning, aggregation, and visualization.

---

## Project Highlights

- Cleaned and analyzed **49,000+** sales records
- Handled missing values and converted data types
- Created a new metric: **Total Sales** (Quantity × Unit Price × (1 - Discount))
- Identified:
  - Top 10 products by total sales
  - Top 10 customers by purchase volume
- Visualized insights using **Matplotlib** (bar and line plots)
